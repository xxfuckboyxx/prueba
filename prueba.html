¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML

HTML (Lenguaje de marcado de hipertexto o HyperText Markup Language por sus siglas en inglés) es un lenguaje descriptivo que especifica la estructura de las páginas web.body para el contenido. "head" para información sobre el documento. "div" división dentro del contenido. "a" para enlaces. "strong" para poner el texto en negrita. "br" para saltos de línea. "H1" "H6" para títulos dentro del contenido. "img" para añadir imágenes al documento.
